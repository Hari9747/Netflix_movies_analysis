Netflix Cleaned Dataset 


Project Overview :

This repository contains the **cleaned Netflix dataset** used for my end-to-end Data Analytics projects.

The original dataset was cleaned using **Python (Pandas) in Jupyter Notebook**. This cleaned dataset was then used for:

- Data Analysis in MySQL
- Interactive Dashboard in Power BI

The cleaning process improved data quality and prepared the dataset for SQL analysis and dashboard creation.

 
Data Cleaning Performed :

The dataset was cleaned using Python (Pandas).

Cleaning steps include:

- Removed duplicate records
- Handled missing values
- Filled unknown values where necessary
- Standardized date format
- Created `year_added`
- Created `month_added`
- Created `main_country`
- Cleaned text formatting
- Prepared dataset for SQL and Power BI
